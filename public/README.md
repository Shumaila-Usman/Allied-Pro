# Public Assets Folder

Place your logo file here with the name: `logo.png`

Supported formats: PNG, JPG, SVG

The logo should be:
- Recommended size: 192px width x 64px height (or proportional)
- Transparent background (PNG) preferred
- High resolution for crisp display

If the logo file is not found, the site will display a gradient placeholder with "ACBS" text.
