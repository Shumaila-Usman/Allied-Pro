# Brand Logos

Place your brand logo images here with these file names:

- `brand-1.png` - SILK B PROFESSIONAL
- `brand-2.png` - MÔND'SUB
- `brand-3.png` - LA
- `brand-4.png` - XANITALIA
- `brand-5.png` - ROMA
- `brand-6.png` - BSC

**Supported formats**: PNG (preferred with transparent background), JPG
**Recommended size**: 200px × 200px (square) or proportional

You can add more brands by adding `brand-7.png`, `brand-8.png`, etc. and updating the brands array in `app/page.tsx`.
